# Janus

![Janus](https://upload.wikimedia.org/wikipedia/commons/7/73/Janus_Bifrons_by_Adolphe_Giraldon.png)

A local dev api proxy server that anybody can use to mock APIs

## Features

* Koa based api server
* API-Blueprint based mocking/documentation publishing/collaboration
* API Proxy that a UI developer needs (need I say more)
